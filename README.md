# FNN_RNN
Simple codes of neural networks

- fnn.py: Feedforward networks
 learning Sin curve
 input(x) -> output(y)

- rnn.py: Recurrent NNs
 learning Sin curve
 input(x) -> output(y)

- rnn_predict.py: predictive learning with RNNs
 learning Lissajous curve
 input(x_t, y_t) -> output(x_(t+1), y_(t+1))
